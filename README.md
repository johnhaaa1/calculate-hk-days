# 港澳停留天数计算工具｜使用教程
一键解析出入境记录，自动计算有效停留天数，适配上海落户审核参考

## 工具简介
本工具用于快速计算**港澳有效停留天数**，仅在浏览器本地运行，不上传任何数据，安全、快捷、隐私保护。

## 使用步骤
1. 打开工具网页
2. 上传你的**出入境记录PDF文件**
3. 工具自动计算并展示：
   - 有效停留总天数
   - 与180天要求的差额
   - 完整行程明细

## 功能说明
- 自动识别出入境时间、剔除无效天数
- 实时显示是否满足上海落户停留要求
- 全程本地处理，不联网、不上传、不收集信息
- 界面简洁，一键出结果

## 隐私保障
- 所有计算**仅在你的浏览器本地完成**
- 不收集、不上传、不存储你的任何个人信息
- 无后台、无日志、无数据上报

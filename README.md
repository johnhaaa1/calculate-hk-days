# 港澳停留天数计算器
适用于国家移民管理局出入境PDF，自动计算有效停留天数（上海落户参考）

## 在线使用
访问地址：(https://johnhaaa1.github.io/calculate-hk-days/)

## 如何获取移民局出入境记录PDF

### 方式一：电脑网页版
1.  打开国家移民管理局政务服务平台官网：https://s.nia.gov.cn/mps/views/query/wgrkcrjlcx.html
2.  点击「登录」，使用身份证号+密码登录（未注册可先完成注册，支持人脸识别）
3.  登录后，选择「出入境记录查询」，设置查询时间段（建议覆盖需计算的完整周期）
4.  点击「查询」，查看完整出入境记录，确认无误后点击「导出」
5.  选择导出格式为「PDF」，保存到本地（导出的PDF即为工具可解析的官方文件）

### 【推荐】方式二：移动端小程序（微信/支付宝）
1.  打开微信/支付宝，搜索并进入官方小程序「移民局12367」（原"移民局NIA"）
2.  点击右下角「我的」，完成登录（支持人脸识别，未注册先完成注册）
3.  返回首页，点击「中国公民服务」→「出入境记录查询」
4.  设置查询时间段，点击「查询」查看完整记录
5.  点击「下载」，输入接收邮箱和提取密码，点击「发送至邮箱」
6.  打开邮箱，下载并解压收到的PDF文件（使用刚才设置的提取密码）

## 使用方法
1.  打开上方在线网址
2.  点击「选择PDF文件」，上传步骤5中保存的移民局官方出入境记录PDF
3.  点击「开始解析并计算」
4.  查看计算结果：总有效天数、落户差额、行程明细

## 详细逻辑补丁
1. 行程闭合（有完整的两两搭配的出境+入境组合，说明您已经回到内地），则正常计算。
2. 行程不闭合（最新的一条数据是出境，说明您还在香港），则用今日的日期减去最后出境的日期得到天数。
3. 中途如果有出境前往他国旅游，出入境口岸均非香港口岸，需要您自行减去这一段您的境外时间（bug正努力修复中，在收集各种情况和口岸名称...如您有类似疑难杂症，欢迎投稿到johnzhou696@gmail.com)。
